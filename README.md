### Focus areas

- **GPU & ML systems**: CUDA/Triton kernels, PyTorch, TensorFlow, NCCL, DDP, multi-GPU and RoCE/InfiniBand. NVIDIA-Certified Associate in AI Infrastructure & Operations and in Generative AI LLMs.
- **LLM & agentic AI**: retrieval-augmented generation, vector search, and MCP integrations that automate cross-team workflows.
- **Computer vision & applied ML**: detection and classification pipelines (YOLO, CNN, transfer learning) built for real-time, low-latency inference.

### Featured projects

- **FlashAttention GPU Kernel (from scratch)**: fused Triton kernel using online-softmax streaming recurrence, cutting peak attention memory from O(N^2) to O(N), about 250x lower at sequence length 8192.
- **[Network Intrusion Detection System](https://github.com/jshingala/NetworkIntrustionSystem)**: FCNN/CNN in TensorFlow tuned with Keras Tuner, reaching 97% accuracy on imbalanced network-traffic data.
- **Ticker Skimmer**: stock predictor fusing Reddit sentiment (NLP) with a bidirectional LSTM time-series model, served through a real-time Gradio UI.
- **Weapon & Face Detection**: two-model computer-vision pipeline (YOLOv11n + CNN via transfer learning from Inception v3) reaching 94% detection accuracy with 87% fewer false positives via dual-model agreement.
